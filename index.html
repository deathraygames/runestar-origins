A Roguelike Retro Game Made for the 7DRL Challenge 2019
A keyboard is required to play.
Press Enter to begin.
Coding by Luke @ Deathray Games
Uses the open source libraries: rote js and rot js
Fonts: "Fix15Mono-Bold" by LambdaHack and "AppleII"

Runestar: Origins

v1.1 for 7DRL 2019
  • Move cardinal: WASD, HJKL, Numpad, Arrows
  • Move diagonal: QWZX, YUBN, Numpad
  • Attack by "bump" movement
  • Ready an ability: 1-9
  • Inventory: I
  • Toss/Drop: T
  • Use/Pickup: Enter
  • Wait: Space
